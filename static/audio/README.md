# Visual project audio

Add the local MP3 files below to enable looping audio when the matching visual is opened:

- `safe-in-your-skin.mp3`
- `need.mp3`
- `marietta.mp3`
- `reconstruction-site.mp3`

Audio starts after 100ms, fades in to 25% volume over 600ms, loops continuously while the lightbox is open, and fades out quickly when it closes.
